# thamizh

The brand landing page for the **thamizh** ecosystem.

- **thamizh** — social, in Tamil
- **thamizhi** — journalism, in Tamil

Built with [Astro](https://astro.build). Vanilla CSS. One page. Two apps.

## Develop

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Stack

- Astro 6
- Geist Sans (CDN)
- Vanilla CSS (no framework, no Tailwind)
- Static SVG asterisk mark

## Design

- Light cream body, red hero
- Direct, present-tense product copy
- One page, two apps
- Static 5-line asterisk logo
