## Spaces

Browse and join constituency spaces across Tamil Nadu and Eelam. Each space is a dedicated discussion area for that constituency.

- Search by name (English or Tamil), district, or number
- Browse Tamil Nadu and Eelam constituencies in a single view
- Pin your frequently used spaces for quick access
- Location selector to switch between Tamil Nadu and Eelam community views
- See last message timestamps and previews for each space

## Feed

A location-based social feed that shows community conversations. The feed adapts to your active location — Tamil Nadu or Eelam.

- Pull-to-refresh for new content
- Infinite scroll to browse older posts
- Rich post rendering with hashtags, mentions, and URLs
- Embedded images and link previews
- Reply, repost, and like counts on each post
- Empty state messaging when no posts are found

## Chats

Private and group messaging for community members.

### Private Chats

- One-on-one conversations with other community members
- Searchable chat list by name or handle
- Conversations sorted by most recent activity
- Read receipts and unread indicators

### Group Chats

- Create new group conversations with community members
- Group member list and member count displayed
- Sender names shown for each message
- Unread badge counts on each group

### Messaging Features

- Message bubbles with sender colours (accent for your messages, grey for others)
- Sender avatars and timestamps on messages
- Keyboard-aware composer with send button
- Empty states prompting conversation starters

## Community

View members of the Tamil Nadu and Eelam communities. Each member profile shows their avatar, display name, handle, and admin status.

- Tamil Nadu Community with active member listings
- Eelam Community with active member listings
- Share invite link via system share sheet
- Admin badges for community coordinators
- Member count displayed at the top

## Profile

Your public profile connected to your Bluesky identity:

- Banner image, avatar, display name, and bio
- Following, followers, and posts counts
- Tabbed feed — Posts, Replies, Media, Likes
- Edit profile — update display name and bio with character limits
- Pull-to-refresh for profile and feed data
- Settings drawer with edit profile, view on Bluesky, and sign out
- Sign out with confirmation and session cleanup

## Constituency Hub

Each constituency has a hub page serving as a dashboard for all governance and community tools.

- Constituency overview with name, number, district, and resident count
- Visual map representation of the constituency
- KPI summary — issues reported, pending items
- 17 Constituency Tools — governance modules including AI assistant, public voice, rights dashboard, participatory budget, anti-caste justice, and more
- 16 Community Tools — mutual aid tools including blood donor registry, tree plantation, disaster relief, education support, and more
- Tamil and English labels for every tool
