title: Core Principles
description: The foundational values that guide Thamizhi's approach to decentralized journalism.
---

Thamizhi is built on five core principles that guide every decision in the platform's design and operation.

## 1. Transparency Over Opacity

Every aspect of Thamizhi is open and auditable:
- **Open source code**: Anyone can inspect how the system works
- **Public verification logs**: All fact-checking decisions are recorded
- **Algorithmic transparency**: AI processing is explainable
- **Community oversight**: Users can challenge any decision

## 2. Community Over Corporation

The platform serves communities, not shareholders:
- **No advertising model**: No incentives to sensationalize
- **No paywalls**: Information access is a right, not a privilege
- **Community governance**: Users shape platform policies
- **Local优先**: Stories affecting communities get priority

## 3. Truth Over Speed

Verification matters more than being first:
- **Multi-layer verification**: AI screening + community review + expert validation
- **Source requirements**: All claims need evidence
- **Correction mechanisms**: Errors are corrected transparently
- **Context preservation**: Stories include full context

## 4. Access Over Exclusivity

Everyone can participate in journalism:
- **Citizen journalism**: Anyone can submit stories
- **Multilingual support**: Content in multiple languages
- **Accessibility-first**: Designed for all abilities

## 5. Privacy Over Surveillance

User protection is paramount:
- **Minimal data collection**: Only what's necessary
- **Encrypted communications**: All sensitive data protected
- **No tracking**: User behavior isn't monitored
- **Right to be forgotten**: Users can delete their data

## Applying These Principles

These principles aren't abstract—they manifest in concrete technical decisions:

- **Groq AI**: Fast, open-source LLM inference
- **GitHub Actions**: Public workflows ensure transparency
- **Cloudflare Workers**: Global distribution ensures access

Every technology choice in Thamizhi can be traced back to these core principles.
