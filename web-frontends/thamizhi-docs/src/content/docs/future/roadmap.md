## Development Phases

### Phase 1: MVP (Current — Q3 2026)

**Goal**: Working prototype with basic scraping, processing, and publishing.

- [x] Documentation site (this site)
- [ ] GitHub Actions scraper for RSS sources (5 sources)

- [ ] Groq AI processing pipeline (extraction + summarization)
- [ ] Poster generation (HTML → screenshot)
- [ ] Cloudflare Workers API (basic CRUD)
- [ ] React Native app MVP (news feed)
- [ ] Citizen submission (basic form)
- [ ] Deploy to Cloudflare Pages + Workers

### Phase 2: Growth (Q4 2026)

**Goal**: Full feature set with community features.

- [ ] All 20+ news sources
- [ ] YouTube scraping + transcription
- [ ] Community review system
- [ ] User reputation system
- [ ] Web app (React Native Web)
- [ ] Search functionality
- [ ] Multi-layer verification (AI + community + expert)
- [ ] Notification system

### Phase 3: Scale (Q1 2027)

**Goal**: Production-ready with AI improvements.

- [ ] Custom fine-tuned LLM (Tamil news model)
- [ ] Government data integration
- [ ] Social media scraping (Twitter, Facebook)
- [ ] Expert reviewer program
- [ ] Analytics dashboard
- [ ] Performance optimization

### Phase 4: Decentralize (Q2 2027+)

**Goal**: True decentralization with P2P distribution.

- [ ] P2P mesh network integration
- [ ] Offline-first capabilities
- [ ] Mesh networking for local news sharing
- [ ] Decentralized identity system
- [ ] Community-governed moderation

## Current Status

✅ **Phase 1 in progress** — Documentation site complete
